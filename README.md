## AdSense가 승인됐는데 광고가 안 보일 때 체크리스트

1. **반영 시간 확인**: 승인 직후에는 광고가 바로 노출되지 않고 보통 수시간~수일(최대 1~2주) 걸릴 수 있습니다.
2. **광고 차단 확장 프로그램 확인**: 브라우저의 AdBlock/uBlock이 켜져 있으면 본인 화면에는 광고가 보이지 않습니다.
3. **광고 자동 게재 설정 확인**: AdSense 대시보드에서 사이트의 `자동 광고`가 켜져 있는지 확인합니다.
4. **스크립트/메타 삽입 확인**: 사이트 `<head>`에 AdSense 스크립트와 `google-adsense-account` 메타가 있어야 합니다.
5. **ads.txt 확인**: 루트 경로의 `ads.txt`가 실제 배포 URL에서도 접근되는지 확인합니다. (`https://도메인/ads.txt`)
6. **크롤링 및 정책 상태 확인**: AdSense의 정책 센터에서 제한 상태가 없는지 확인합니다.

현재 이 저장소에는 AdSense 스크립트와 메타 태그, `ads.txt`가 이미 포함되어 있으므로, 승인 직후라면 시간 지연 이슈일 가능성이 큽니다.
